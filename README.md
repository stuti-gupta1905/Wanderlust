# WanderLust

A full-stack Airbnb-inspired web application where users can browse listings, create and manage properties, book stays, see maps and leave reviews.
Compatible with several devices like desktop, phone, tablet, etc.

## Brief Overview
Once you signup/login into your account, you can view/edit/delete your profile, create/edit/delete listings/reviews, book properties and get notified if they're available. You can view the location of the listing on a map. You can view the listing owner's/review author's profile as well. You can search through key-words and get results if they match with the title/location/description of a listing. You can redirect to the terms and privacy policy page from the footer.
#### All the user related options are available on the top-right corner (in the navbar).
You can seamlessly upload images for your listing or profile picture in png/jpg/jpeg format.

## Features
- User authentication
- Signup/Login/Logout
- Property listings
- Booking system with date validations
- Reviews and ratings
- Search functionality
- Cloudinary image uploads
- Mapbox integration
- Pofile, MyBookings, GuestBookings pages
- Listing and Review editing as well as deletion after user authorization
- etc.
